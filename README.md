# Movie Recommendation System (Data Science Project)

## Project Overview
This project implements a **content-based movie recommendation system** with a strong focus on **data science fundamentals** including exploratory data analysis, statistical analysis, feature engineering, and similarity-based recommendations.

## Objectives
- Understand movie and user rating data
- Perform exploratory data analysis (EDA)
- Apply statistical reasoning and hypothesis testing
- Build a content-based recommendation system using cosine similarity
- Extract insights and interpret results

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scipy
- Jupyter Notebook
